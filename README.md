# Note Reviewer — Obsidian Plugin

[![Buy Me a Coffee](https://img.shields.io/badge/Buy%20Me%20a%20Coffee-gabrielsilvestri-yellow?logo=buy-me-a-coffee)](https://buymeacoffee.com/gabrielsilvestri)

Review and **declutter your Obsidian vault** one note at a time. Note Reviewer shows each note as a card — you decide to keep it or send it to trash. Progress is saved between sessions, so you can review even the largest vaults at your own pace.

## Features

- **One note at a time** — focused review without overwhelm
- **Keyboard-first** — K / → keep, D / ← trash, Z undo, Space keep
- **Session persistence** — close and reopen anytime, pick up where you left off
- **Native Obsidian trash** — uses Obsidian's `.trash/` folder, fully recoverable
- **Undo support** — restore the last deleted note instantly
- **Markdown preview** — see the full note content rendered before deciding
- **Open in Obsidian** — jump to the note with one click while reviewing
- **Progress tracking** — see how many notes you've reviewed across all sessions

## Installation

### Community Plugins (recommended)

1. Open Obsidian → Settings → Community Plugins
2. Search for **"Note Reviewer"**
3. Click Install, then Enable

### Manual (BRAT)

1. Install [BRAT](https://github.com/TfTHacker/obsidian42-brat)
2. Add `gabrielsilvestri/obsidian-note-reviewer` as a beta plugin

## Usage

1. Click the **layers icon** in the left ribbon, or run `Open Note Reviewer` from the command palette (Ctrl+P)
2. Review notes one by one:

| Key | Action |
|-----|--------|
| K or → or Space | Keep note |
| D or ← | Send to trash |
| Z | Undo last deletion |

3. Close anytime — your progress is saved. Reopen to continue from where you left off.
